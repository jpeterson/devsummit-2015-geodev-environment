"Software Craftsmanship"

This term is used to express the idea that a developer is more than just someone programming a machine to achieve a desired output. Developers, like craftsmen, develop skills and become proficient using the tools of craft. You don't see professional photograhers using disposable cameras, or lumberjacks using hacksaws.
